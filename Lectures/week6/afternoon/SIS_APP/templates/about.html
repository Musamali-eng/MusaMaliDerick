About Student Information System

Version 2.0 - Modern Student Management Platform

📖 Overview

The Student Information System (SIS) is a comprehensive web application designed to streamline the management of student records. Built with modern technologies, it provides an intuitive interface for administrators to register, track, and manage student information efficiently.

✨ Key Features

  • ✅ Student Registration with validation
  • ✅ View all registered students
  • ✅ Search students by name
  • ✅ Edit and update student information
  • ✅ Delete student records
  • ✅ View student profiles
  • ✅ Responsive design for all devices

🛠️ Technology Stack

Flask SQLAlchemy SQLite HTML5 CSS3 Jinja2

📊 Database Schema

Students Table
  • id (Primary Key)
  • student_name
  • registration_number (Unique)
  • email (Unique)
  • programme
  • is_active
  • created_at
